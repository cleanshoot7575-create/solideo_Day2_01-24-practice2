# 스마트 여행 예약 시스템

실시간 교통편 검색 및 최저가 비교 서비스

## 주요 기능

### ✈️ 교통편 검색
- **출발지/도착지** 입력
- **날짜/시간** 선택
- **교통수단** 선택 (비행기, 기차, 버스)
- **승객 수** 설정

### 💰 실시간 가격 비교
- 최저가 자동 표시
- 일반 가격 평균 계산
- 총 비용 산정 (승객 수 반영)

### 🗺️ 관광 명소 지도
- Google Maps 연동
- 목적지별 주요 관광지 표시
- 각 명소 상세 정보 제공

### 📊 실시간 정보
- 잔여 좌석 정보
- 취소된 항공편 자동 제외
- 운항 회사별 시간표
- 소요 시간 및 도착 시간

## 실행 방법

### 방법 1: 로컬 서버 (추천)
```bash
python3 -m http.server 8080
```
그 다음 브라우저에서 접속:
```
http://localhost:8080/travel-booking-app.html
```

### 방법 2: 직접 열기
브라우저에서 파일을 직접 열기:
```
travel-booking-app.html
```

## 지원 경로

현재 시스템에서 지원하는 구간:

1. **서울 ↔ 제주도**
   - 비행기 (김포-제주)

2. **서울 ↔ 부산**
   - KTX (서울역-부산역)
   - SRT (수서역-부산역)
   - 고속버스
   - 비행기 (김포-김해)

3. **부산 ↔ 제주도**
   - 비행기 (김해-제주)

## 기술 스택

- **HTML5**: 구조
- **CSS3**: 스타일링 (그라데이션, 애니메이션)
- **JavaScript**: 동적 기능
- **Google Maps API**: 지도 및 관광명소

## 주요 특징

✅ 반응형 디자인 (모바일/태블릿/데스크탑)
✅ 실시간 검색 시뮬레이션
✅ 최저가 자동 하이라이트
✅ 잔여 좌석 실시간 표시
✅ 취소편 자동 필터링
✅ 관광명소 인터랙티브 지도
✅ 가격 비교 (최저/평균)
✅ 다양한 교통수단 지원

## 사용 예시

1. 출발지에 "서울" 입력
2. 도착지에 "제주도" 입력
3. 날짜와 시간 선택
4. 승객 수 입력
5. "실시간 검색하기" 버튼 클릭
6. 결과 확인 및 지도에서 관광명소 탐색
